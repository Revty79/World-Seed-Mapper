# 08 — Seed Noise Generation

## Goal
Add a simple procedural terrain seed generator for quickly blocking out a world.

## Requirements
- Add a way to generate a starting heightmap using noise/fractal-like generation.
- User should be able to trigger regeneration.
- Include at least a small number of adjustable parameters, such as:
  - seed
  - scale
  - roughness/intensity
- Generated output should feed into the same editable heightmap system.

## Deliverables
- Procedural terrain seed generation
- Basic controls for generation settings
- Integration with existing heightmap workflow

## Notes
- Keep it simple and usable.
- This is a seed tool, not a final realism system.
