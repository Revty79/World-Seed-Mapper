# 05 — Local Project Save / Load

## Goal
Allow the user to save and load projects locally without any backend.

## Requirements
- Add a serializable project format.
- Support saving the current project to a local file.
- Support loading a previously saved project file.
- Include basic validation/error handling for invalid files.
- Include a visible Project area or controls for save/load/new project.

## Deliverables
- Working save flow
- Working load flow
- New project/reset flow
- Minimal validation messaging

## Notes
- JSON format is acceptable.
- Do not introduce a backend.
