# 04 — Label Tools

## Goal
Add label creation and editing for named map features.

## Requirements
- Users can place a label on the map.
- Users can set/edit:
  - label text
  - approximate position
- Users can select an existing label.
- Selected label details should appear in the inspector.
- Labels should be part of the serializable project model.

## Deliverables
- Basic label placement workflow
- Label list or selection support
- Label editing UI
- Labels rendered on top of the map

## Notes
- Dragging labels is nice if easy, but not required for first pass.
- Fancy typography is not required yet.
