# 00 — Foundation Scaffold

## Goal
Create the initial application scaffold for World Seed Mapper.

## Requirements
- Set up a Next.js app with TypeScript and Tailwind CSS.
- Create a visually strong but minimal landing/editor shell.
- Create an initial app layout with a clear split for:
  - top bar
  - left tool area
  - center editor area
  - right inspector/preview area
- Add placeholder navigation/sections for:
  - Map
  - Layers
  - Tools
  - Globe
  - Project
- The app should feel like a purpose-built creative tool, not a generic admin panel.

## Deliverables
- Running app scaffold
- Global styles and layout foundation
- Placeholder editor shell page
- Clean component structure for future editor modules

## Notes
- No real editor behavior yet.
- Focus on structure, naming, visual direction, and maintainability.
